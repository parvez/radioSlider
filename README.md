# jQuery Radio Slider 1.0

This repository contains the source code for jQuery Radio Slider 1.0

## Usage

Need to have two lists as follows:

	<div id="mydiv">
      <ul>
      	<li>Option 1</li>
      	<li>Option 2</li>
      	<li>Option 3</li>
      </ul>
  	</div>

To initialize:

 	$("#mydiv").radioSlider();
